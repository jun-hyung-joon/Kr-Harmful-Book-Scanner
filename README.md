# (청소년)유해간행물 조회 시스템

## 개요
카메라로 도서 뒷면의 ISBN 바코드를 스캔하여 청소년 유해간행물 여부를 즉시 판별하는 정적 웹 애플리케이션입니다. 별도의 프로그램 설치 없이 웹 주소 접속만으로 즉시 이용 가능합니다.

## 사용 방법
1. **서비스 접속**: [https://jun-hyung-joon.github.io/Kr-Harmful-Book-Scanner/](https://jun-hyung-joon.github.io/Kr-Harmful-Book-Scanner/) 링크를 통해 접속합니다.
2. **카메라 권한 허용**: 최초 접속 시 브라우저에서 요청하는 **카메라 접근 권한을 허용**해야 스캔 기능이 정상 작동합니다.
3. **바코드 스캔**: 카메라 화면 중앙에 도서 뒷면의 ISBN 바코드를 비추면 조회가 완료되며 결과 화면으로 전환됩니다.

## 데이터 안내
* **기준 시점**: 내장된 데이터(`data.csv`)는 **2026년 6월 자료**까지 적용되었습니다. 
* **데이터 출처**: 간행물윤리위원회 (청소년)유해간행물 목록표

## 기술 스택
* HTML5 / CSS3 / Vanilla JavaScript
* @zxing/library (바코드 인식 엔진)
* PapaParse (CSV 파싱 라이브러리)
